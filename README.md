
#  CodeManiacs

Dashboard: Displays leaderboard and rewards.

Rule Based Kitty: Navigate a kitty using commands (move forward, turn left, turn right) to reach goals, avoid obstacles, and collect keys within a step limit.

Blockly Low Code: Use blocks to create logical sequences, like Scratch, to navigate to a goal. Reveals JavaScript code upon completion.

Dungeon Coder: Split screen with IDE and 2D dungeon. Control character with code to complete objectives.

Daily Challenges: Complete daily tasks to earn XP.

Multiplayer Code: Compete against another player in coding challenges. Best of 3 rounds wins, displayed as "Win" or "Lose" after each round.


## Installation

Install my-project with npm

```bash
  npm install 

  npm run dev
```
    
## Features

Dashboard
Leaderboard: Displays top players based on XP.
Rewards: Shows available rewards and XP needed to redeem them.
Player Status: Indicates whether players are active, paused, or on vacation.

Rule Based Kitty
Move Commands: Use commands like move forward, turn left, and turn right.
Obstacles and Keys: Navigate through obstacles and collect keys.
Step Limit: Complete the task within a specified number of steps.
Goal Position: Reach the designated goal position.

Blockly Low Code
Code Blocks: Use blocks such as move forward, turn right, turn left, if, if-else, and while.
Logical Sequences: Connect blocks to create logical code sequences.
Goal Navigation: Navigate character to the goal position.
JavaScript Reveal: Reveals underlying JavaScript code upon level completion.

Dungeon Coder
Split Screen: IDE on one side, 2D dungeon game on the other.
Character Control: Control character using code typed in the IDE.
Objectives: Complete in-game objectives using code.
Phaser JS: Game developed using Phaser JS framework.

Daily Challenges
XP Rewards: Complete challenges to earn XP.
Difficulty Levels: Challenges of varying difficulty levels.
Progress Tracking: Track progress and XP earned from daily tasks.

Multiplayer Code
Competitive Coding: Compete against another player in real-time.
Timed Rounds: Best of 3 rounds to determine the winner.
IDE for Each Player: Each player has an IDE to write and run their code.
Problem Solving: Solve displayed coding problems within the time limit.
Win/Lose Display: Displays "Win" for the winner and "Lose" for the loser after each round.
Final Results: Shows the overall winner at the end of the game.
